# Контент-компаньон: YouTube, новости, сериалы, рецепты

Статус на 2026-07-12: минимальный intent-router для `chat`, `open_vass`,
`youtube_search` и `youtube_watch` реализован в фазе 7.3. Новости, сериалы и
рецепты остаются следующими расширениями того же контракта.

Единый механизм — **intent-роутер** перед основным LLM-ответом: лёгкий
незаземлённый Gemini side-call возвращает строго типизированное действие.
Клиент объявляет capability `supportsExternalActions`; старые клиенты не
получают action events и продолжают работать как обычный чат.

## YouTube

В 7.3 видео не встраивается в Vass или overlay. Сервер передаёт по SSE только
`youtube_search {query}` либо `youtube_watch {videoId}`. Мобильный клиент ещё
раз валидирует payload и сам строит один из двух разрешённых universal URL:

- `https://www.youtube.com/results?search_query=...`;
- `https://www.youtube.com/watch?v=...`.

Android открывает URL через системный handler: установленный YouTube либо
браузер. Произвольный URL от модели не принимается, package-specific API и
`AccessibilityService` не используются. Конкретный watch допускается только
для уже указанного пользователем валидного 11-символьного video ID; иначе
команда безопасно остаётся поиском.

YouTube Data API v3 и встроенный `react-native-youtube-iframe` не нужны для
этого внешнего сценария. Если позже появится отдельный fullscreen-плеер Vass,
его квоты и ToS принимаются отдельным продуктовым решением.

## Новости

Gemini + Search grounding уже умеет (доказано на погоде) — для «почитай
новости» добавляется intent, который просит LLM собрать сводку из свежих
новостей и озвучивает её по предложениям. Опция: RSS-агрегатор на бэкенде
для стабильных источников.

## Сериалы/фильмы

TMDB API (бесплатный, есть русская локализация, отдаёт «где посмотреть» через
watch-providers) — intent `series_search`: «найди сериал про …» → карточка +
голосовое описание.

## Рецепты (сценарий №1 презентации)

Режим пошагового чтения с паузами — intent `recipe_read`, состояние «шаг N»,
команды «дальше»/«повтори»/«стоп». Источник: Search grounding или
Spoonacular/Edamam.
