# Контент-компаньон: YouTube, новости, сериалы, рецепты

Единый механизм — **intent-роутер** перед основным LLM-ответом (расширение
уже работающего preamble-паттерна): лёгкий Gemini-вызов классифицирует
реплику → `{intent: chat|youtube_play|news_read|series_search|recipe_read,
params}`.

## YouTube

`react-native-youtube-iframe` (официальный IFrame-плеер, соответствует ToS).
Поиск — YouTube Data API v3 (бесплатная квота 10k units/день с головой
хватает). Голосом: «включи видео про …» → intent → поиск → автоплей первого
результата → голосовое управление («стоп», «дальше», «громче»).

Ограничение ToS: фоновое проигрывание в IFrame запрещено — см.
[risks.md](./risks.md).

## Новости

Gemini + Search grounding уже умеет (доказано на погоде) — для «почитай
новости» добавляется intent, который просит LLM собрать сводку из свежих
новостей и озвучивает её по предложениям. Опция: RSS-агрегатор на бэкенде
для стабильных источников.

## Сериалы/фильмы

TMDB API (бесплатный, есть русская локализация, отдаёт «где посмотреть» через
watch-providers) — intent `series_search`: «найди сериал про …» → карточка +
голосовое описание.

## Рецепты (сценарий №1 презентации)

Режим пошагового чтения с паузами — intent `recipe_read`, состояние «шаг N»,
команды «дальше»/«повтори»/«стоп». Источник: Search grounding или
Spoonacular/Edamam.
